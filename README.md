# Churn_Prediction


# 🚀 Customer Churn Prediction Web Application  

A professional **Machine Learning Web Application** built to predict whether a customer is likely to leave a company based on customer behavior and business-related factors.  

This project combines **Machine Learning + Flask + Frontend Development** to create a smart predictive system with an interactive UI.

---

## 🌐 Live Project  

🔗 Local Server:  
http://127.0.0.1:5000  

---

## 📌 Project Overview  

Customer churn prediction helps businesses identify customers who may stop using their services.  

This application allows users to input customer details and predicts churn probability using a trained Machine Learning model.

The project focuses on:  

✅ Data Preprocessing  
✅ Feature Engineering  
✅ Model Training  
✅ Prediction System  
✅ Frontend Integration  
✅ Flask Deployment

## ⚙ Installation  

Clone the repository  

```bash
git clone <your-github-repository-link>
```

Go to project folder  

```bash
cd Customer-Churn-Prediction
```

Install dependencies  

```bash
pip install -r requirements.txt
```

Run the Flask server  

```bash
python app.py
```

Open browser  

```bash
http://127.0.0.1:5000
```
